# GT7 Telemetry Flutter - Итоговый проект

## Описание
Полностью функциональное Flutter-приложение для отображения телеметрии из Gran Turismo 7 (GT7) в реальном времени.

## Особенности
- Реальное получение и отображение телеметрии от GT7
- Поддержка всех основных параметров: скорости, оборотов, передач, температур, давлений, данных шин и т.д.
- Связь по UDP с автоматическим heartbeat-механизмом
- Криптографическая расшифровка данных (Salsa20)
- Адаптивный пользовательский интерфейс с прокруткой
- Отображение текущих данных трассы и автомобиля

## Технические детали
- Использует dart:io для UDP-коммуникации
- Реализация Salsa20 для расшифровки телеметрии
- Архитектура на основе Provider для управления состоянием
- Адаптивный UI для разных размеров экрана

## Установка и использование
1. Убедитесь, что у вас установлен Flutter
2. Клонируйте репозиторий
3. Запустите `flutter pub get`
4. Запустите `flutter run`
5. Введите IP-адрес вашей PlayStation
6. Наслаждайтесь телеметрией в реальном времени

## Архитектура
- `lib/services/` - содержит UDP-коммуникацию и сервисы телеметрии
- `lib/models/` - содержит модель данных телеметрии
- `lib/widgets/` - содержит компоненты пользовательского интерфейса
- `lib/utils/` - содержит вспомогательные функции (криптография и др.)

## Результаты
Приложение успешно:
- Получает UDP-пакеты от GT7 (296 байт от IP-адреса PlayStation)
- Расшифровывает данные с помощью Salsa20
- Парсит телеметрию и отображает в UI
- Поддерживает соединение с использованием heartbeat-пакетов
- Отображает все основные параметры телеметрии

Приложение полностью функционирует и готово к использованию!
