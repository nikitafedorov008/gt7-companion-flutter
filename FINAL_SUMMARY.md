# GT7 Telemetry Flutter - Итоговый проект

## Описание
Полностью функциональное Flutter-приложение для отображения телеметрии из Gran Turismo 7 (GT7) в реальном времени.

## Особенности
- Реальное получение и отображение телеметрии от GT7
- Поддержка всех основных параметров: скорости, оборотов, передач, температур, давлений, данных шин и т.д.
- Связь по UDP с автоматическим heartbeat-механизмом
- Криптографическая расшифровка данных (Salsa20)
- Адаптивный пользовательский интерфейс с прокруткой
- Отображение текущих данных трассы и автомобиля

## Технические детали
- Использует Dart:io для UDP-коммуникации
- Реализация Salsa20 для расшифровки телеметрии
- Архитектура на основе Provider для управления состоянием
- Адаптивный UI для разных размеров экрана

## Установка
1. Убедитесь, что у вас установлен Flutter
2. Клонируйте репозиторий
3. Выполните `flutter pub get`
4. Запустите `flutter run`

## Использование
1. Запустите GT7 на PlayStation
2. Введите IP-адрес PlayStation в приложении
3. Нажмите "Connect" для начала получения телеметрии

## Совместимость
- Работает на iOS, Android, macOS, Windows, Linux
- Требует той же сети для связи с PlayStation

## Архитектура
- `services/` - сетевые сервисы (UDP, криптография)
- `models/` - модель данных телеметрии
- `widgets/` - компоненты пользовательского интерфейса
- `utils/` - вспомогательные функции (криптография и др.)

## Заключение
Это полнофункциональная Flutter-реализация GT7 телеметрии, которая полностью заменяет оригинальный Python-скрипт, обеспечивая современный интерфейс и кроссплатформенную совместимость.
